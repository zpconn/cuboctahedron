\title{A Formal Reduction of the Cuboctahedron Omnihedral Billiard Problem}
\author{Zach Conn}
\date{\today}

\begin{document}

\begin{abstract}
We give a self-contained account of a Lean-checkable reduction for the
cuboctahedron omnihedral billiard problem.  A nonsingular billiard orbit
visiting each face exactly once is unfolded to a straight line in a reflected
development.  After choosing the first face to be \(X_+\), the possible face
orders are represented by \(97{,}297{,}200\) opposite-face pair words, together
with six sign choices in the identity-linear case.

The central structural theorem is a holonomy normal form.  It scans a pair
word by tracking the parity of the three square-pair reflections and a
projective triangular shadow.  For every valid started pair word \(w\), the
linear return map is the identity if and only if the reduced triangular shadow
\(\red(w)\) is empty.  The nonempty-shadow branch is handled by forced-axis
obstructions, while the empty-shadow branch is handled by exact rational
linear inequalities, a necessary positive-denominator condition, and Farkas
certificates.

The finite coverage part is stated as an explicit theorem: every
nonidentity rank and every identity rank with each sign mask is assigned a
Lean-checked obstruction.  External programs may generate proposed data, but
Lean checks the corresponding exact integer and rational certificates.
\end{abstract}

\section{Statement of the Problem}

A billiard trajectory in a polyhedron moves along straight line segments and
reflects specularly at faces.  We exclude singular impacts at edges and
vertices.

\begin{definition}[Nonsingular omnihedral orbit]
Let \(P\subset\R^3\) be a polyhedron with face set \(\F\).  A periodic
billiard orbit in \(P\) is \emph{omnihedral} if, during one period, it meets
every face of \(P\) exactly once.  It is \emph{nonsingular} if every impact
point lies in the relative interior of the face it meets.
\end{definition}

\begin{theorem}[Target theorem]
The cuboctahedron has no nonsingular periodic billiard orbit which meets each
of its fourteen faces exactly once in one period.
\end{theorem}

The proof is formalized in Lean 4.  This article describes the mathematical
objects and finite coverage theorem used by the formalization.  The Lean
repository is
\[
  \url{https://github.com/zpconn/cuboctahedron}.
\]

\section{The Cuboctahedron}

We use the rational coordinate model
\[
P=\{(x,y,z)\in\R^3:
 |x|\le 1,\ |y|\le 1,\ |z|\le 1,\
 \pm x\pm y\pm z\le 2\}.
\]
The six square faces are
\[
X_\pm:\ \pm x=1,\qquad
Y_\pm:\ \pm y=1,\qquad
Z_\pm:\ \pm z=1,
\]
and the eight triangular faces are
\[
T_{\epsilon_1\epsilon_2\epsilon_3}:
\epsilon_1x+\epsilon_2y+\epsilon_3z=2,
\qquad \epsilon_i\in\{\pm1\}.
\]

Each face \(F\) lies in a plane
\[
n_F\dotp p=c_F
\]
with \(n_F\in\Q^3\) and \(c_F\in\Q\).  Reflection in this plane is the affine
map
\[
r_F(p)=p-2\,\frac{n_F\dotp p-c_F}{n_F\dotp n_F}\,n_F.
\]
Thus every reflection used below has exact rational coefficients.

\begin{definition}[Opposite-face pairs]
Two faces are in the same opposite-face pair if their supporting planes are
parallel.  The seven pairs are denoted
\[
x,\ y,\ z,\ d_{111},\ d_{11\bar 1},\ d_{1\bar 1 1},\
d_{\bar 1 11}.
\]
The first three are square pairs.  The last four are triangular pairs, named
by projective normal directions.
\end{definition}

Opposite faces have the same linear reflection part and differ only in their
affine translation term.  This is why the proof first classifies words in
opposite-face pairs and then adds signs where needed.

\section{Unfolding}

\begin{definition}[Face itinerary]
A period-14 face itinerary is a function
\[
  s:\{0,\ldots,13\}\to\F.
\]
It is \emph{started} if \(s(0)=X_+\), and it is \emph{omnihedral} if it is a
bijection from \(\{0,\ldots,13\}\) to the fourteen faces.
\end{definition}

For an itinerary \(s\), let
\[
A_s=r_{s(13)}\circ\cdots\circ r_{s(0)}
\]
be the affine return map of the unfolded development.  Write
\[
A_s(p)=M_s p+b_s.
\]
The order convention in the Lean code is equivalent to multiplying the
corresponding reflection matrices in the order in which the reflected copies
are crossed.

\begin{definition}[Unfolded witness]
An \emph{unfolded witness} for an itinerary \(s\) is a line
\[
\ell(t)=p+tv,\qquad v\ne0,
\]
in the reflected development such that:
\begin{enumerate}[leftmargin=2em]
\item \(\ell\) crosses the developed copy of \(s(i)\) at the \(i\)-th impact;
\item every crossing lies in the relative interior of the corresponding
  developed face;
\item \(A_s\) carries the line to itself and preserves its direction, so
  \(M_s v=v\).
\end{enumerate}
\end{definition}

\begin{lemma}[Unfolding criterion]
A nonsingular billiard orbit with face itinerary \(s\) exists if and only if
there is an unfolded witness for \(s\).
\end{lemma}

\begin{proof}
At each impact, reflect the next copy of the polyhedron across the impact
face rather than reflecting the velocity vector.  The reflected outgoing
segment becomes the continuation of the incoming segment, so the billiard path
unfolds to a straight line.  Nonsingularity is exactly the assertion that the
crossing points stay in relative face interiors.  Periodicity says that the
affine return map identifies the developed line with itself.  Reflecting the
copies back reverses the construction and restores the equal-angle reflection
law.
\end{proof}

\begin{lemma}[Starting-face reduction]
If a nonsingular omnihedral period-14 orbit exists, then there is one whose
first face is \(X_+\).
\end{lemma}

\begin{proof}
An omnihedral period hits \(X_+\) exactly once.  Cyclically reindex the
period so that this hit is first.
\end{proof}

It is therefore enough to rule out started unfolded witnesses.

\section{Pair Words, Ranks, and Sign Masks}

\begin{definition}[Started pair word]
A \emph{started pair word} is a length-13 word in the seven opposite-face
pairs recording the pair of each face after the initial face \(X_+\).  Such a
word is \emph{valid} if the multiplicities are
\[
x:1,\qquad y:2,\qquad z:2,\qquad
d_{111},d_{11\bar 1},d_{1\bar 1 1},d_{\bar 1 11}:2.
\]
The set of valid started pair words is denoted \(\mathcal W\).
\end{definition}

The cardinality is
\[
|\mathcal W|=\frac{13!}{2^6}=97{,}297{,}200.
\]

\begin{definition}[Rank]
A \emph{ranking} is a fixed bijection
\[
\unrank:\{0,\ldots,97{,}297{,}199\}\to\mathcal W.
\]
The inverse map is denoted \(\rank\).  A statement proved for every rank is
therefore a statement for every valid started pair word.
\end{definition}

The Lean formalization defines a lexicographic rank/unrank map and proves that
it is a bijection.  The choice of order is not mathematically significant; it
is an addressing scheme for the finite set \(\mathcal W\).

\begin{definition}[Realizing a pair word]
A started itinerary \(s\) \emph{realizes} a valid started pair word \(w\) if,
for each \(i=0,\ldots,12\), the face \(s(i+1)\) belongs to the opposite-face
pair \(w_i\).
\end{definition}

\begin{definition}[Sign mask]
In a started itinerary, the signs of the six noninitial opposite-face pairs
are independent after the first face \(X_+\) is fixed.  A \emph{sign mask} is
an element of
\[
\Masks=\{\pm1\}^6.
\]
For a valid pair word \(w\) and a mask \(m\in\Masks\), write
\[
  s(w,m)
\]
for the signed face itinerary obtained by choosing the face in each pair
according to \(m\), when this notation is used in the identity-linear branch.
\end{definition}

\section{The Holonomy Normal Form}

For a valid pair word \(w\), let \(M(w)\) be the linear part of the unfolded
return map for the corresponding pair reflections.  Since opposite faces have
the same linear reflection part, \(M(w)\) depends only on the pair word, not
on the sign mask.

\begin{definition}[Linear holonomy type]
A valid pair word \(w\) is \emph{identity-linear} if \(M(w)=I\), and
\emph{nonidentity-linear} if \(M(w)\ne I\).
\end{definition}

The product convention for a pair word
\[
w=(w_0,\ldots,w_{12})
\]
is
\[
  R(w_0)R(w_1)\cdots R(w_{12})R(x),
\]
where the final \(R(x)\) is the started \(X_+\) pair reflection.

\begin{definition}[Square parity]
The \emph{square parity} is an element
\[
  p=(p_x,p_y,p_z)\in(\Z/2\Z)^3.
\]
Scanning a square-pair letter \(x,y,z\) toggles the corresponding coordinate
of \(p\).  Scanning a triangular-pair letter leaves \(p\) unchanged.
\end{definition}

\begin{definition}[Triangular letters]
The four triangular pair directions are represented by projective normal
letters
\[
  \mathcal T=\{111,\ 11\bar 1,\ 1\bar 1 1,\ \bar 1 11\}.
\]
Changing the sign of all three coordinates gives the same projective letter.
\end{definition}

\begin{definition}[Parity action on triangular letters]
Let \(p=(p_x,p_y,p_z)\in(\Z/2\Z)^3\), and write a triangular letter in
projective form as \((1,\eta,\zeta)\) with
\(\eta,\zeta\in\{\pm1\}\).  The action of \(p\) sends it to the projective
letter represented by
\[
\bigl(1,\ (-1)^{p_x+p_y}\eta,\ (-1)^{p_x+p_z}\zeta\bigr).
\]
This is the effect of commuting accumulated square reflections past a
triangular reflection and then normalizing projectively.
\end{definition}

\begin{definition}[Reduced triangular shadow]
Scan the fourteen pair factors
\[
  w_0,w_1,\ldots,w_{12},x
\]
from left to right.  Maintain a square parity \(p\), initially zero, and a
stack of triangular letters, initially empty.  On a square-pair letter, update
\(p\).  On a triangular-pair letter \(t\), push the acted letter \(p\cdot t\)
onto the stack, except that if it is equal to the current top of the stack,
delete the current top instead.  The final stack, read in chronological order,
is the \emph{reduced triangular shadow} \(\red(w)\).
\end{definition}

\begin{theorem}[Identity classifier]
For every valid started pair word \(w\),
\[
  M(w)=I
  \quad\Longleftrightarrow\quad
  \red(w)=[].
\]
\end{theorem}

\begin{proof}
The scan is designed so that the product of pair reflections factors as
\[
  M(w)=T(w)S(w),
\]
where \(S(w)\) is the diagonal matrix determined by the final square parity
and \(T(w)\) is the product of the triangular shadow reflections.  Validity of
\(w\) implies that the scan sees \(x,y,z\) each an even number of times after
the final started \(x\) factor is included; hence \(S(w)=I\).

Adjacent equal triangular reflections multiply to the identity, so reducing
the triangular stack does not change \(T(w)\).  It remains to show that a
nonempty reduced triangular word cannot have product \(I\).

For a triangular normal \(n\in\{\pm1\}^3\), the reflection matrix is
\[
  H_n=I-\frac{2}{3}nn^T.
\]
Let
\[
  K_n=3H_n=3I-2nn^T\in M_3(\Z).
\]
Modulo \(3\), \(K_n\equiv nn^T\).  A product of nonempty reduced triangular
letters gives, modulo \(3\), a product of rank-one matrices.  Adjacent
projective letters are distinct, and the corresponding dot products are
nonzero in \(\mathbb F_3\), so the mod-3 product is a nonzero scalar multiple
of an outer product.  It is therefore nonzero.  If the rational triangular
product were \(I\), the scaled integer product of length \(m>0\) would be
\(3^mI\), which is zero modulo \(3\), a contradiction.
\end{proof}

\begin{remark}
An external exact scan over all \(97{,}297{,}200\) ranks found
\[
2{,}468{,}088
\]
empty reduced shadows and
\[
94{,}829{,}112
\]
nonempty reduced shadows.  These numbers are diagnostics and are not used as
proof; the theorem above is the trusted classifier.
\end{remark}

\section{Obstructions for the Two Branches}

\subsection{The nonidentity branch}

By the identity classifier, \(\red(w)\ne[]\) implies \(M(w)\ne I\).

\begin{definition}[Nonidentity case killed]
For a rank \(r\), let \(w=\unrank(r)\).  The rank \(r\) is
\emph{nonidentity-killed} if
\[
M(w)\ne I
\]
implies that no started omnihedral itinerary realizing \(w\) has an unfolded
witness.
\end{definition}

\begin{lemma}[Forced-axis condition]
Let \(A(p)=Mp+b\) be the affine return map for an itinerary with \(M\ne I\).
If there is an unfolded witness with line \(p_0+\R v\), then \(v\ne0\),
\[
Mv=v,
\]
and there is a scalar \(\lambda\in\R\) such that
\[
  (I-M)p_0=b-\lambda v.
\]
\end{lemma}

\begin{proof}
The return map preserves the line and its direction, so \(Mv=v\).  The point
\(A(p_0)=Mp_0+b\) lies on the same line, so
\[
Mp_0+b=p_0+\lambda v
\]
for some \(\lambda\), which is equivalent to the displayed equation.
\end{proof}

The nonidentity certificates used by the formalization are exact rational
certificates showing that this forced line geometry cannot satisfy the
required itinerary.  A certificate may prove, for example, that there is no
compatible fixed direction, that the forced axis misses the interior of
\(X_+\), that a crossing direction has the wrong sign, or that a prescribed
impact point is outside its face interior.

\subsection{The identity branch}

By the identity classifier, \(\red(w)=[]\) implies \(M(w)=I\).  The affine
return map is then a translation
\[
  A(p)=p+b.
\]
For a nondegenerate translation witness, the unfolded line has direction
parallel to \(b\).  Since the first face is \(X_+\), the starting point may be
written
\[
  p_0=(1,y,z).
\]
For a fixed pair word \(w\) and sign mask \(m\), all face-interior and
crossing-order conditions become strict rational linear inequalities in
\(y,z\).

\begin{definition}[Impact denominator]
For an internal impact \(i\), let the corresponding unfolded face plane be
\[
  u_i\dotp p=d_i.
\]
The crossing time along the translation direction is
\[
  t_i=\frac{d_i-u_i\dotp p_0}{u_i\dotp b}.
\]
The quantity
\[
  D_i(w,m)=u_i\dotp b
\]
is the \emph{impact denominator} at \(i\).
\end{definition}

\begin{definition}[GoodDirection]
For an identity-linear pair word \(w\) and sign mask \(m\), the pair
\((w,m)\) satisfies \emph{GoodDirection} if
\[
  D_i(w,m)>0
\]
for every internal impact \(i=1,\ldots,13\).
\end{definition}

\begin{lemma}[GoodDirection is necessary]
If an identity-linear signed itinerary has an unfolded witness, then it
satisfies GoodDirection.
\end{lemma}

\begin{proof}
The unfolded line must cross each internal face in forward time and from the
prescribed side.  With the chosen orientation, the numerator in the displayed
formula for \(t_i\) has the prescribed sign; a nonpositive denominator would
make the required forward crossing impossible or oriented incorrectly.
\end{proof}

\begin{definition}[Translation case killed]
For a rank \(r\) with \(w=\unrank(r)\) and a sign mask \(m\), the signed case
\((r,m)\) is \emph{translation-killed} if
\[
M(w)=I
\]
implies that no started omnihedral itinerary realizing \(w\) with sign mask
\(m\) has an unfolded witness.
\end{definition}

\begin{definition}[All-Good translation coverage]
An \emph{all-Good translation coverage proof} is a proof that, for every rank
\(r\) and mask \(m\), if \(M(\unrank(r))=I\) and \((\unrank(r),m)\) satisfies
GoodDirection, then the signed case \((r,m)\) is translation-killed.
\end{definition}

By the necessity of GoodDirection, all-Good translation coverage implies
ordinary translation coverage for all masks, including those that fail
GoodDirection.

\subsection{Farkas certificates}

\begin{definition}[Strict rational system in two variables]
A strict rational system in variables \(y,z\) is a finite family of
inequalities
\[
  a_jy+c_jz+e_j>0,\qquad a_j,c_j,e_j\in\Q.
\]
\end{definition}

\begin{lemma}[Farkas certificate]
Suppose a strict rational system admits rational multipliers \(q_j\ge0\), not
all zero, such that
\[
  \sum_j q_ja_j=0,\qquad
  \sum_j q_jc_j=0,\qquad
  \sum_j q_je_j\le0.
\]
Then the system has no real solution.
\end{lemma}

\begin{proof}
If \(y,z\) solved all inequalities, multiplying by \(q_j\) and summing would
give
\[
0<
\sum_jq_j(a_jy+c_jz+e_j)
=
\Big(\sum_jq_ja_j\Big)y+
\Big(\sum_jq_jc_j\Big)z+
\sum_jq_je_j
\le0,
\]
a contradiction.
\end{proof}

\begin{definition}[Source-indexed row]
A \emph{source-indexed row} is a linear inequality together with a label
identifying the geometric condition from which it was derived, such as the
starting point lying in \(X_+\), an ordering condition between impacts, or an
impact point lying in a specified face interior.
\end{definition}

Source-indexed certificates allow generated data to name the geometric origin
of a Farkas row rather than relying only on a row number in a large matrix.
The soundness theorem checks both that the named source supplies the displayed
row and that the rational multipliers give the Farkas contradiction.

\subsection{Walsh expressions}

\begin{definition}[Walsh expression]
Let \(s_1,\ldots,s_6\in\{\pm1\}\) be the sign-mask variables.  A
\emph{quadratic Walsh expression} is a rational expression of the form
\[
\alpha+\sum_i\beta_i s_i+\sum_{i<j}\gamma_{ij}s_is_j,
\qquad
\alpha,\beta_i,\gamma_{ij}\in\Q.
\]
\end{definition}

For a fixed pair word, each impact denominator \(D_i(w,m)\) is a quadratic
Walsh expression in the six sign variables.  This is because the unfolded
normal \(u_i\) and the translation vector \(b\) are affine functions of the
signs, and \(D_i=u_i\dotp b\) is their dot product.

\begin{definition}[Walsh subcube obstruction]
A \emph{subcube} is obtained by fixing some of the six signs and leaving the
others free.  A Walsh subcube obstruction is a proof that a denominator, or a
nonnegative rational linear combination of denominators, is nonpositive on a
subcube.  Such an obstruction rules out every mask in the subcube, because
GoodDirection requires all internal denominators to be positive.
\end{definition}

\section{The Finite Coverage Theorem}

\begin{definition}[Semantic coverage proof]
A \emph{semantic coverage proof} consists of:
\begin{enumerate}[leftmargin=2em]
\item a proof that every rank in \(\{0,\ldots,97{,}297{,}199\}\) denotes a
  valid started pair word;
\item a proof that every sign mask is one of the \(64\) elements of
  \(\Masks\);
\item a proof that every rank is nonidentity-killed;
\item a proof that every rank and every mask form a translation-killed case.
\end{enumerate}
The word ``semantic'' means that the theorem states the impossibility of the
corresponding unfolded witness directly.  It does not require the theorem
statement to expose the raw certificate object used internally.
\end{definition}

\begin{definition}[Generated proof]
A proof or certificate is \emph{generated} if an external program writes down
its proposed data.  It is trusted only after Lean checks a theorem whose proof
uses that data.  The external program is not part of the trusted base.
\end{definition}

\begin{proposition}[Coverage implies no started unfolded witness]
If there is a semantic coverage proof, then no started omnihedral itinerary
has an unfolded witness.
\end{proposition}

\begin{proof}
Let \(s\) be a started omnihedral itinerary with an unfolded witness.  Forget
signs to obtain a valid started pair word \(w\), and let \(r=\rank(w)\).
By the identity classifier, either \(\red(w)\ne[]\), hence \(M(w)\ne I\), or
\(\red(w)=[]\), hence \(M(w)=I\).

In the first case, the nonidentity-killed part of coverage contradicts the
unfolded witness.  In the second case, the itinerary determines a sign mask
\(m\), and the translation-killed part of coverage contradicts the unfolded
witness.
\end{proof}

\begin{theorem}[Conditional formal theorem]
If there is a semantic coverage proof for all ranks and masks, then the
cuboctahedron has no nonsingular omnihedral periodic billiard orbit.
\end{theorem}

\begin{proof}
Assume such a billiard orbit exists.  By the starting-face reduction, cyclically
reindex it so that it starts on \(X_+\).  By the unfolding criterion, it gives
a started omnihedral itinerary with an unfolded witness.  This contradicts the
previous proposition.
\end{proof}

\section{Formal Verification Boundary}

The Lean development separates two kinds of work.

First, it proves reusable mathematical theorems: the cuboctahedron geometry,
the unfolding criterion, the finite rank/unrank enumeration, the holonomy
normal form, GoodDirection necessity, and Farkas soundness.

Second, it must supply the finite semantic coverage proof.  External scripts
may search for source-indexed Farkas certificates, Walsh subcube obstructions,
forced-axis certificates, or other exact witnesses.  Lean must check each
certificate or family theorem before it contributes to coverage.

The proof code is constrained to exact arithmetic.  In particular, the final
formal proof must not rely on floating-point arithmetic, numerical tolerances,
custom axioms, `sorry`, `admit`, `unsafe`, or native-code evaluation.

\section{Validation}

The repository contains a conditional theorem with the shape
\begin{verbatim}
theorem Cuboctahedron.conditional_cuboctahedron_no_omnihedral
    (coverage : ExhaustiveGeneratedCoverage) :
    Not (exists o : BilliardOrbit14,
      o.Nonsingular /\ o.Periodic /\ o.TouchesEachFaceExactlyOnce)
\end{verbatim}
where `ExhaustiveGeneratedCoverage` is the Lean package corresponding to the
finite coverage theorem above.

For the conditional theorem, the intended checks are:
\begin{verbatim}
lake build
grep -R "sorry\|admit\|axiom\|native_decide\|unsafe" Cuboctahedron || true
lake env lean Cuboctahedron/PrintConditionalAxioms.lean
\end{verbatim}

When the unconditional coverage theorem is installed, the final axiom check is:
\begin{verbatim}
lean Cuboctahedron/PrintAxioms.lean
\end{verbatim}
